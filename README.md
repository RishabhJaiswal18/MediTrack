## Overview
MediTrack is a console-based healthcare management system built using pure Core Java.

## Features
- Patient & Doctor Management
- Appointment Booking & Cancellation
- Billing & Tax Calculation
- CSV & Serialization based persistence
- Java Streams analytics

## Concepts Covered
- JVM Internals
- OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
- Design Patterns
- Generics & Collections
- File I/O
- Concurrency basics
- Streams & Lambdas

## How to Run
javac MediTrack.java
java MediTrack
